# Car-Recommendation-System
Our Car Recommendation System helps users find the perfect car based on brand, fuel type, transmission, and cost range. Using the K-Nearest Neighbors (KNN) algorithm, the model analyzes user preferences and suggests the best matches by comparing similar vehicles, providing a personalized and efficient car selection experience.
